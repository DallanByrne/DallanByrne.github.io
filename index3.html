My Work

Learn Hiragana

'Learn Hiragana' is a single page site which teaches the japanese hiragana characters. All presently used characters are displayed on a table. When the user hovers their mouse over any character it plays a gif tracing the character. Pronunciations of the letters is played when the characters are clicked.

The site also provides a quiz for the user to test their knowledge. On selecting a column they wish to be quized on they are presented with a php designed test. The test includes mathing the romanized characters to images and vice versa.

Further additions to be added include a page for the Katakana alphabet, simple japanese words to learn using hiragana and possibly a Learn Kanji page.
